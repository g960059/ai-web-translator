## Detailed Description

**Fibo** (named after Fibonacci) is an AI translation extension built for researchers and engineers who read technical content with formulas and code.

### Why Fibo instead of Google Translate?

When you translate a math paper with Google Translate or DeepL, the equations break.

Fibo **protects LaTeX, MathML, and code blocks before translating**. The formulas stay intact; only the surrounding text is translated. arXiv preprints, Wikipedia math articles, GitHub documentation — read them all without mangled equations.

### Trust & Privacy

🔒 **No data collected** — Translation text is sent directly to OpenRouter's API, never to Fibo's developer. No analytics, no tracking, no developer-run servers.

🔓 **Open source (MIT)** — All code is public on GitHub. Inspect it yourself.

💰 **No subscription — pay per use** — Use your OpenRouter API key and pay only for what you translate. A typical Wikipedia article costs $0.01–0.05. Cost estimates shown before you translate.

### Getting Started (2 minutes)

1. Create a free account at [OpenRouter](https://openrouter.ai) (a platform providing access to AI models from Google, Meta, and others)
2. Get your API key (credit card required for billing)
3. Enter the key in Fibo's settings
4. Press the translate button on any page!

**No Fibo account needed.** Your API key is stored locally in your browser only.

### Features

- ✅ Full protection for LaTeX / MathML / code blocks
- ✅ Automatic full-page translation (main content first)
- ✅ Translation cache (instant on revisit)
- ✅ Translation / Bilingual / Original — 3 display modes
- ✅ Hover to show original text
- ✅ 3 model presets + custom model from OpenRouter's full catalog
- ✅ Alt+T keyboard shortcut
- ✅ Per-site translation blocklist

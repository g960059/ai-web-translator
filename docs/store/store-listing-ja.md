# Chrome Web Store 掲載内容（日本語）

## 名前
Fibo

## 概要（132文字以内）
arXivの数式もGitHubのコードも壊さないAI翻訳。ページ全体をワンクリックで翻訳し、LaTeX・MathML・コードブロックを100%保持。月額不要、1ページ約3円。

## 詳細説明

論文の数式、ちゃんと残ってますか？

arXivの論文をページ翻訳したら数式がぐちゃぐちゃ。技術ドキュメントを翻訳したらコードブロックが崩壊。そんな経験、ありませんか？

Fiboは、数式・コード・HTML構造を100%保持したまま、AIがページ全体を自然な日本語に翻訳するChrome拡張です。

🔬 数式もコードも、そのまま

LaTeX、MathML、コードブロック、テーブル構造——すべて原文のまま保持して翻訳します。arXiv、Wikipedia、GitHub、技術ブログなど、理系コンテンツの翻訳に最適化されています。

他の翻訳ツールでは壊れてしまう数式が、Fiboではそのまま表示されます。もう原文と壊れた翻訳結果をイライラしながら見比べる必要はありません。

⚡ ワンクリックで全文翻訳

ページを開いてAlt+T（Mac: Option+T）を押すだけ。本文を優先的に翻訳し、スクロールに応じて残りを自動で翻訳します。選択範囲だけの翻訳も右クリックから。

・翻訳 / 対訳 / 原文の3モード切替
・ホバーで原文をすぐ確認
・翻訳キャッシュで再訪問時は待ち時間ゼロ

💰 月額不要。1ページ約3円

サブスク不要。OpenRouterのAPIキーを設定するだけで使えます（従量課金）。
Wikipediaの長い記事を全訳しても約3円。月に100ページ翻訳しても約300円。

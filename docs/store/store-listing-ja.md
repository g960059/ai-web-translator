# Chrome Web Store 掲載内容（日本語）

## 名前
Fibo

## 概要（132文字以内）
数式やコードを壊さず、技術ページを母国語で読める翻訳拡張。arXiv・Wikipedia・技術ドキュメントに最適。

## 詳細説明

**Fibo**（フィボナッチに由来）は、数式やコードを含む技術文書を意味を損なわずに翻訳する、研究者・エンジニアのためのAI翻訳拡張機能です。

### 他の翻訳ツールとの違い

Google 翻訳や DeepL で数学の論文を翻訳すると、数式が壊れませんか？

Fibo は LaTeX・MathML・コードブロックを**保護してから翻訳**します。数式はそのまま、周囲のテキストだけを自然な日本語に。arXiv のプレプリント、Wikipedia の数学記事、GitHub のドキュメント — すべて数式を崩さずに読めます。

### 安心して使える

🔒 **データを収集しません** — 翻訳テキストは OpenRouter の API に直接送信され、Fibo の開発者がデータを受け取ることはありません。アナリティクスやトラッキングも一切なし。

🔓 **オープンソース（MIT）** — コードはすべて公開されています（GitHub）。

💰 **月額なし・使った分だけ** — OpenRouter の API キーで、実際の使用量に応じた支払い。Wikipedia 1記事あたり約1〜5円。翻訳前にコスト見積もりを表示します。

### はじめ方（2分で完了）

1. [OpenRouter](https://openrouter.ai) で無料アカウントを作成（AI モデルへのアクセスを提供するプラットフォームです）
2. API キーを取得（クレジットカード登録で即時利用可能）
3. Fibo の設定画面にキーを入力
4. 翻訳したいページでボタンを押すだけ！

**Fibo のアカウント登録は不要です。**API キーはブラウザ内にのみ保存されます。

### 主な機能

- ✅ LaTeX / MathML / コードブロックの完全保護
- ✅ ページ全体の自動翻訳（本文を優先）
- ✅ 翻訳キャッシュ（再訪問時は即座に表示）
- ✅ 翻訳 / 対訳 / 原文の3モード切り替え
- ✅ ホバーで原文を表示
- ✅ 3つのモデルプリセット + OpenRouter 全モデルからカスタム選択
- ✅ Alt+T で翻訳開始（キーボードショートカット）
- ✅ サイトごとの翻訳除外設定

### こんなサイトで使えます

📐 arXiv プレプリント・数学/物理の論文
💻 GitHub ドキュメント・MDN・Read the Docs
📚 Wikipedia の専門記事
🔬 技術ブログ・Stack Overflow
